# EduManage Logo Assets

## Logo Variants

### Primary Logo
- `edumanage-logo.svg` - Stacked layout (icon + wordmark)
- `edumanage-horizontal.svg` - Horizontal layout for headers
- `edumanage-dark.svg` - Dark mode version with white text

### Icon Only
- `edumanage-icon.svg` - Full icon for app usage
- `edumanage-favicon.svg` - Simplified icon for small sizes (16px-32px)

## Color Palette
- **Primary Teal**: #0EA5A4
- **Secondary Navy**: #0F172A  
- **White**: #FFFFFF

## Usage Guidelines

### Minimum Sizes
- Primary logo: 120px width minimum
- Icon only: 16px minimum
- Favicon: 16px-32px optimal

### Applications
- **Headers**: Use horizontal layout
- **Login screens**: Use primary stacked layout
- **Favicons**: Use simplified favicon version
- **Dark backgrounds**: Use dark mode variant

## Technical Notes
- All SVGs are optimized and production-ready
- Scalable from 16px to 1000px+
- Clean code with no metadata or raster images
- System fonts used for maximum compatibility
